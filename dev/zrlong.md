## Information

1. row
  - alphabet
  - sheng：声母可以不变。
  - yun：对应的韵母。
2. table 
  - yun：如何单独打出韵母。
  - bianma：击键顺序。
  
## Convert

row 每 8 个为一个 row
下面的意义何在？
```js
{ alphabet: ';' },
```

以:为间隔，右侧为按键key，左侧为对应韵母yun。
key -> alphabet uppercase
yun -> table/yun 排序

table
yun：按照首字母放在一个 table
删去 i 开头，没有这样拼音构成的字。
